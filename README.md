## How to upload to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder, not the zip itself.
3. Go to repository **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from branch**.
5. Select `main` branch and `/root` folder.
6. Save.

## Resume link
Replace:

`assets/resume.pdf`

with your real resume file. Keep the same filename so all website buttons keep working.

## Contact form setup
The contact form currently uses this placeholder:

`https://formspree.io/f/YOUR_FORM_ID`

To make it work:
1. Go to Formspree.
2. Create a free form.
3. Copy your endpoint.
4. In `index.html`, search for `YOUR_FORM_ID`.
5. Replace it with your real Formspree form ID.

Example:

`https://formspree.io/f/abcdwxyz`
